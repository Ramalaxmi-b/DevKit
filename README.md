# DevKit Chrome Extension

DevKit is a Chrome extension designed to provide useful tools for web developers. It includes features like window resizing, link checking, and technology analysis.

## Features

- **Window Resizer**: Resize the current Chrome window to predefined or custom dimensions.
- **Link Checker**: Check the status of links on a web page for broken links.
- **Tech Analyzer**: Analyze the technologies used on a web page.

## Installation

1. Clone the repository:
git clone https://github.com/Ramalaxmi-b/DevKit.git


2. Navigate to `chrome://extensions/` in your Chrome browser.

3. Enable Developer mode (toggle switch usually located in the top right corner).

4. Click on "Load unpacked" and select the directory where you cloned the repository.

## Usage

- **Window Resizer**: Open the extension popup, enter custom dimensions or select a predefined size from the dropdown, and click "Resize Window".
- **Link Checker**: Open the "Link Checker" tab in the extension popup, enter a page URL, and click "Check Links".
- **Tech Analyzer**: Open the "Tech Analyzer" tab, enter a page URL, and click "Analyze Tech" to view the technologies used.
